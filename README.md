Travel-matching application of group 9.1 for the course "Fundamentals and Methods of Computer Science for Business Studies"

How to install:

# clone the repository
git clone https://github.com/Shiggytora/ComputerScience.git

# navigate to the repository
cd ComputerScience

# create a virtuell environment
python -m venv .venv

# install dependency
pip install -r requirements.txt

# run streamlit/start the app
streamlit run streamlit_app.py
